# Forum Notifications

## Events
- Reply to your thread: notify the thread author when someone replies (email + in-app).
- Mention in post: notify mentioned worker(s) with a link to the post.
- Upvotes on your post: notify when a post crosses configurable upvote thresholds.

## Preferences
- Email: send transactional emails via Supabase functions.
- In-app: show bell icon alerts within worker dashboard.
- None: allow opt-out to respect notification fatigue.
