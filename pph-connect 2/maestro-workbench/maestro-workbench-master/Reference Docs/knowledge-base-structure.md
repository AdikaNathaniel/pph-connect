# Knowledge Base Structure

## Categories
- **Getting Started**: onboarding, access, first-day checklists.
- **Best Practices**: productivity tips, community guidelines, workflow hacks.
- **Troubleshooting**: common errors, resolution steps, escalation paths.
- **Policies**: compliance, privacy, leave policies.

## Articles
- Rich text Markdown content with metadata (author, updated date, views).
- Each entry linked to a category; support search tags and highlights.
