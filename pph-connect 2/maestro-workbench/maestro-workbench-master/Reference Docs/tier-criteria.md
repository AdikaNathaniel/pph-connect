# Tier Criteria Plan

## Tier0 → Tier1
- **Tasks completed**: 500+ lifetime tasks.
- **Quality score**: 95%+ 90-day rolling average.
- **Assessments**: pass Tier1 knowledge assessment.
- **Compliance**: zero quality violations in the past 90 days.

## Tier1 → Tier2
- **Tasks completed**: 1000+ lifetime tasks.
- **Quality score**: 98%+ 90-day rolling average.
- **Assessments**: pass Tier2 exams plus domain expert interview.
- **Leadership**: positive peer/team lead reviews.

## Demotion Criteria
- Quality score below tier threshold for 30 consecutive days.
- ≥3 quality violations within the last quarter.
- Failure to complete mandatory refresh training.
